# Bioprocess Optimization UI/UX Builder

A web application for defining and optimizing constraints and objectives in bioprocess engineering, built with **JavaScript**, **React**, **Vite**, **Tailwind CSS**, and **Recharts**.

Developed by using Test-Driven Development (TDD) principles to ensure modular and maintainable code.

---

## Features Overview

### **1.Constraint Builder**
Define bioprocess constraints with an intuitive token-based interface.
